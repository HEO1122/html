

a 태그
==> 다른 사이트 혹은 다른 문서로 이동할 때 사용하는 태그

	형식 ]
		
		<a href="이동할사이트 혹은 문서">
		
	참고 ]
		절대경로 : 시스템 혹은 사이트의 특정 문서의 위치를 말한다.
		상대경로 : 현재 보고있는 문서를 기준으로 위치를 찾아가는 방식
					
					.	: 현재 문서와 동일한 위치(폴더)를 의미
					..	: 현재 문서의 위치(폴더)에서 상위 위치(폴더)
					
				우리의 경우 day02/Test02.html 에서
							doc/day02.txt
							==>
								../../doc/day02.txt
								
	참고 ]
		class	:	.	- 중복해서 써줘도 된다.
		id		: 	#	- 유일한 값으로 써주는 것이 원칙이다.
		
		
		
	**********
	속성중 클래스, 아이디, 네임 속성의 부여 의미
		
		클래스(class)	-	css 에서 사용할 목적으로 부여한다.
		아이디(id)		-	자바스크립트에서 특정 태그를 선택할 목적으로 사용한다.
		네임(name)		-	입력태그에만 붙여주는 속성이다.
							입력태그에 입력된 내용을 다른 페이지로 전송할 목적으로 붙여준다.


입력태그
==> 데이터를 입력받는 용도의 태그
	
	input
	==> 단일행 입력태그
		inline 태그
		예 ] 이름, 아이디,...
		
		속성 ]
			type
				==> 이 입력태그가 어떤용도인지를 알려주는 속성
				속성값 ]
					text, file, password, date,... 
					button, submit, reset,...
			name
			id
			
	textarea
	==> 다중행 입력태그
		block 태그
		예 ] 게시글,..
		
		속성 ]
			name
			id
			cols, rows ==> 입력받을 수 있는 행과 열의 수를 지정하는 옵션
			
*****
입력태그는 반드시 name 속성값을 부여해주는 것이 원칙이다.
입력태그는 form 태그로 감싸주는 것이 좋다.

form 태그
==> 웹 문서에 어떤 데이터를 입력하고 그 결과값을 가져오기 위해서 사용
	==> 감싸여진 입력태그들의 입력 내용을 읽어 올 수 있다.
	
	속성 ]
		method
		==> 데이터를 전달하는 방식
			종류 ]
				GET		==> URL 표시되는 방식
				POST	==> 감춰서 보내는 방식
		
		action
		name
		id
